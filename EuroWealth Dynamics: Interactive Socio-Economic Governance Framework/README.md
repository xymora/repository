# EuroWealth Dynamics
## Interactive European Income Governance (2021-2025)

### Project Overview
This project implements a high-level Data Governance strategy applied to European economic indicators. It features an interactive Dashboard that allows for real-time data exploration through dynamic UI controls.

### Governance Metrics
* **Wealth Distribution:** Multi-national analysis of annual income.
* **Fiscal Responsibility:** Tax Compliance Score as a quality governance indicator.
* **Market Segmentation:** Classification by Income Brackets (Low to Ultra-High).

### Technical Stack
* **Language:** Python 3.9+
* **Visualization:** Plotly Premium (Dynamic HTML5)
* **Ingestion:** Remote CSV Ingestion with NumPy-based data synthesis.
