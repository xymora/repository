# Amazon Sales Strategic Insights - Data Governance Practice (2021)

## Overview
This project implements a high-level **Data Governance Framework** focused on Data Quality, Metadata Management, and Strategic Sales Analysis. It utilizes historical Amazon sales data (2005-2021) to establish a "Single Source of Truth" (SSOT) for executive decision-making.

## Data Governance Pillars
1. **Data Stewardship:** Automated validation of schema and business rules.
2. **Quality Management:** Monitoring of completeness, accuracy, and consistency.
3. **Data Lifecycle:** Automated ingestion from public CSV repositories and secure processing.

## Project Structure
- `practice_governance.py`: Core engine for data ingestion, cleaning, and professional dashboarding.
- `requirements.txt`: Python dependencies.
- `.gitignore`: Standard exclusions for data science environments.
- `LICENSE`: MIT License.

## Usage
1. Install dependencies: `pip install -r requirements.txt`
2. Run the script: `python practice_governance.py`
